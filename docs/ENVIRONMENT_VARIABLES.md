# Environment Variable Reference

| Variable | Purpose |
| --- | --- |
| `SUNAVE_ENV` | Runtime environment (`development`, `staging`, `production`) |
| `SUNAVE_LOG_LEVEL` | Log verbosity |
| `SUNAVE_TENANCY_MODE` | Multi-tenant mode configuration |
| `API_PORT` | API container port mapping |
| `WEB_PORT` | Web container port mapping |
| `DATABASE_URL` | External PostgreSQL connection |
| `REDIS_URL` | External Redis connection |
| `AI_GATEWAY_URL` | External AI gateway endpoint |
| `OLLAMA_BASE_URL` | External Ollama endpoint |
| `STT_SERVICE_URL` | External STT service endpoint |
| `TTS_SERVICE_URL` | External TTS service endpoint |
| `AI_DEFAULT_CAPABILITY` | Default capability token (not a model name) |
| `AI_REQUEST_TIMEOUT_MS` | AI request timeout |
